# Changelog

All notable changes to YesBill are documented here.

## v1.0.0 — March 2026

### New Features
- **Service Tracking** — 5 service types: Home Delivery, Visit-based, Utility, Subscription, Payment
- **Calendar View** — Daily/monthly/yearly tracking with visual attendance grid
- **AI Bill Generation** — OpenAI, Anthropic, and Google Gemini support
- **Ask AI Chatbot** — Persistent conversation history with multi-model support
- **Floating AI Agent** — Intercom-style assistant with action confirmations
- **Bill Payment Tracking** — Mark bills paid with payment method and notes
- **Analytics Dashboard** — Service spend breakdown + AI usage metrics
- **Multi-Provider AI** — Model selection with OpenAI, Anthropic, Google
- **User Profile** — Cover image, avatar, bio, display name
- **Notification System** — Per-type preferences with real-time updates
- **Secure Account Management** — Email/password change with confirmation flow

### Technical
- FastAPI backend with Supabase (PostgreSQL + Auth + RLS)
- React + Vite frontend with Tailwind CSS
- Supabase Realtime for live notification updates
- GitHub Pages deployment with GitHub Actions
