# Support

YesBill has a built-in support system that lets you raise bugs, request features, or report AI-related issues directly from the app.

## Opening Support

![YesBill support screen](/img/screenshots/Supoort-001.png)

Click **Support** in the sidebar to open the support centre. From here you can choose the type of request you want to submit.

## Raise a Bug

![Raise a bug form](/img/screenshots/Support-Raise-a-Bug-001.png)

Use this form to report something that isn't working correctly:

1. **Describe the bug** — What happened? What did you expect to happen?
2. **Steps to reproduce** — How can we replicate it?
3. **Severity** — How badly does this affect your usage?
4. Click **Submit** — the report is sent to the YesBill team

:::tip
The more detail you provide, the faster we can fix it. Include the page you were on and any error messages you saw.
:::

## Request a Feature

![Feature request form](/img/screenshots/Support-Feature-request-001.png)

Have an idea that would make YesBill better?

1. **Feature title** — Short description of what you want
2. **Describe the feature** — How should it work? What problem does it solve?
3. **Priority** — How important is this to you?
4. Click **Submit**

## AI Feature Issue

![AI feature issue form](/img/screenshots/Support-AI-Feature-Issue-001.png)

Use this form specifically for issues related to the AI features (Ask AI, AI bill summaries, AI agent):

1. **Which AI feature?** — Select the feature that had the issue
2. **Describe the issue** — What response was wrong or unhelpful?
3. Click **Submit**

This helps us improve the AI prompts and context accuracy.

## Response Times

The YesBill team reviews support tickets regularly. You'll receive a response via the email address on your account.
