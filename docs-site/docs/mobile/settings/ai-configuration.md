# AI Configuration

Configure your AI provider, API key, and preferred model for Ask AI and the AI Agent.

## AI Settings Screen

<PhoneFrame src="/img/screenshots/mobile/AI-Settings-Screen-01.jpeg" alt="AI settings screen" />

Open **Settings → AI Configuration** to set up your AI provider.

## Supported Providers

YesBill supports three AI providers. You need to bring your own API key.

| Provider | Where to Get a Key |
|----------|-------------------|
| **OpenAI** | [platform.openai.com](https://platform.openai.com) |
| **Anthropic** | [console.anthropic.com](https://console.anthropic.com) |
| **Google** | [aistudio.google.com](https://aistudio.google.com) |

## Provider Settings

<PhoneFrame src="/img/screenshots/mobile/Provider-Settings-Screen-01.jpeg" alt="Provider settings screen" />

1. Select your **AI Provider** from the dropdown.
2. Enter your **API Key** for that provider.
3. Select a **Model** from the list (models vary by provider).
4. Tap **Save**.

### Available Models

- **OpenAI**: GPT-4o, GPT-4o Mini
- **Anthropic**: Claude 3.5 Sonnet, Claude 3 Haiku
- **Google**: Gemini 1.5 Pro, Gemini 1.5 Flash

## Appearance Settings

<PhoneFrame src="/img/screenshots/mobile/Appearance-Settings-Screen-01.jpeg" alt="Appearance settings screen" />

In **Settings → Appearance**, you can configure:

- **Theme** — Light, Dark, or System Default
- **Color Scheme** — Choose an accent color (if available)

:::tip
GPT-4o Mini and Gemini 1.5 Flash are significantly cheaper than their Pro counterparts while still being excellent for bill Q&A. They're a great starting point if you want to minimise API costs.
:::

:::info
Your API key is stored securely on your device and only sent directly to the AI provider — never to YesBill's servers.
:::
